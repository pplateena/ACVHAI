ЛР №3. “Згорткові нейронні мережі”

Мета роботи
Набуття практичних навичок побудови згорткових нейронних мереж, заміни та підлаштування їх складових частин. 
Завдання
Натренувати CNN-мережу для простої задачі класифікації. Варіанти завдань:
MNIST
CIFAR-10
ImageNet
Dogs vs Cats
Власний датасет
Використати transfer learning - У архітектурі передбачити використання попередньо натренованої backbone-мережі для екстракції ознак.
Мережа екстракції ознак повинна містити банк фільтрів, що не змінюється у процесі навчання.
Не зачіпаючи параметри попередньо натренованої backbone-мережі, перебудувати її таким чином, щоб фільтри з банку накладалися на вхідне зображення через добуток у просторі ДПФ.
Провести оцінку продуктивності оригінальної моделі, та моделі із ДПФ-екстрактором ознак, використовуючи релевантні метрики для задачі:
ROC-AUC
f1-score
MAPE
При оцінці моделей, використати початкове значення генератора випадкових чисел (random seed), для прозорості оцінки обох моделей.
Порівняти швидкість навчання обох моделей при використанні
фільтрів малої розмірності
фільтрів великої розмірності
